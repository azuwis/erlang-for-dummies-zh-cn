软件安装

Erlang

现在就来看看怎么安装Erlang。

在Linux发行版上安装

Fedora 19/20

运行以下命令即可安装Erlang。

$ su -c 'yum install erlang'

安装后用erl命令即可打开Erlang Shell。

在Windows上安装

用Windows须自行下载安装程序。你可以从官方网站下载。

以17.0版为例

32位Windows
选择OTP 17.0 Windows 32-bit Binary File
64位Windows
选择OTP 17.0 Windows 64-bit Binary File

若下载很慢，不妨试试这个镜像。

以17.0版为例

32位Windows
选择17.0那行的win32
64位Windows
选择17.0那行的win64

安装完，以17.0版为例，可以从开始菜单 -> 所有程序 -> Erlang OTP 17 -> Erlang打开Erlang Shell。

Erlang Shell

打开Erlang Shell后，第一行会显示Erlang的版本信息，第二行是空行，第三行会显示Shell的版本信息。因为每次打开总是会显示这三行，所以以后都会忽略这三行。而1> 表示这后面是Shell里的第1次输入。

Erlang/OTP 17 (erts-6.0) [async-threads:10]

Eshell V6.0  (abort with ^G)
1>

输入io:format("Hello, world!~n").，按回车，会显示Hello, world!。现在会看到2> ，表示这后面是Shell里的第2次输入。

1> io:format("Hello, world!~n").
Hello, world!
ok
2>

输入halt().，按回车后，就退出Erlang Shell了。

1> io:format("Hello, world!~n").
Hello, world!
ok
2> halt().

GNU Emacs

推荐使用Emacs编辑Erlang代码。虽然使用别的文本编辑器编辑Erlang也不会有问题，但是Erlang官方安装包里已经提供了编辑Erlang代码必要的Emacs模块，可以直接使用，就不用费劲去找了。

在Linux发行版上安装

Fedora 19/20

运行以下命令即可安装Emacs。

$ su -c 'yum install emacs'

安装完用emacs命令即可打开Emacs。

在Windows上安装

用Windows须自行下载Emacs。在GNU Emacs首页上，找到nearby GNU mirror，并点击进入。进入后，找到windows目录，以24.3版为例，选择emacs-24.3-bin-i386.zip。若下载很慢，不妨试试这个镜像。

下载完只需要解压到合适的目录就可以了。找到解压出来的目录下的bin目录，双击runemacs.exe即可运行。为方便以后打开，不妨在runemacs.exe上右键点发送到 -> 桌面快捷方式。推荐把启始目录设置成你放代码的目录，在桌面的快捷方式图标上右键点属性，在快捷方式页可以修改起始目录。

用Windows须自行配置Emacs。按The Erlang mode for Emacs里Setup on Windows一节的说明设置.emacs文件。若不知道当前设置的HOME环境变量，可以进入Erlang Shell查看，运行os:getenv("HOME").，假如结果是false，那就是没有设置HOME环境变量。
